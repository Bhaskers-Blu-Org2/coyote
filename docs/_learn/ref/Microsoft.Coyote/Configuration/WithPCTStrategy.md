# Configuration.WithPCTStrategy method

Updates the configuration to use the PCT scheduling strategy during systematic testing. You can specify the number of priority switch points, which by default are 10.

```csharp
public Configuration WithPCTStrategy(bool isFair, uint numPrioritySwitchPoints = 10)
```

| parameter | description |
| --- | --- |
| isFair | If true, use the fair version of PCT. |
| numPrioritySwitchPoints | The nunmber of priority switch points. |
