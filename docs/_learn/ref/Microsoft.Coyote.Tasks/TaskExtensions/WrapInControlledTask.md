# TaskExtensions.WrapInControlledTask method (1 of 2)

Returns a dummy controlled [`Task`](../TaskType) that wraps this uncontrolled Task.

```csharp
public static Task WrapInControlledTask(this Task @this)
```

## Remarks

The returned dummy controlled [`Task`](../TaskType) does not actually take control of the uncontrolled Task during systematic testing, so this method should only be used to cross an interface boundary where a controlled [`Task`](../TaskType) must be temporarily converted into an uncontrolled Task and then coverted back to a controlled [`Task`](../TaskType).
